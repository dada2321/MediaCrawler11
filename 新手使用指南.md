# 🔰 MediaCrawler 新手完全使用指南

> 📝 **专为完全小白准备的详细教程**  
> 即使您从未接触过编程，也能轻松上手使用！

## 🎯 开始之前

### 这个工具能做什么？
MediaCrawler 是一个**自媒体数据采集工具**，可以帮您：
- 📱 从小红书、抖音、快手、B站等平台采集公开数据
- 📊 分析热门内容和用户评论
- 📈 生成数据统计图表
- 💾 导出数据进行进一步分析

### 您需要准备什么？
- 💻 一台电脑（Windows/Mac/Linux 都可以）
- 🌐 稳定的网络连接
- ⏰ 大约 15-20 分钟的设置时间

---

## 📋 第一步：环境准备

### 1.1 安装 uv 包管理工具（推荐）

**什么是 uv？**
- uv 是最新的 Python 包管理工具，比 pip 更快更稳定
- 能自动管理 Python 版本和虚拟环境
- 原项目强烈推荐使用

**安装 uv：**

**Windows 用户：**
```bash
# 方法一：使用 PowerShell（推荐）
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

# 方法二：使用 pip 安装
pip install uv
```

**Mac 用户：**
```bash
# 使用 Homebrew（推荐）
brew install uv

# 或使用 curl
curl -LsSf https://astral.sh/uv/install.sh | sh
```

**Linux 用户：**
```bash
# 使用 curl
curl -LsSf https://astral.sh/uv/install.sh | sh
```

**验证安装：**
```bash
uv --version
```
如果显示版本号，说明安装成功！

### 1.2 安装 Node.js（部分平台需要）

如果您要爬取**抖音**或**知乎**，需要安装 Node.js：

1. 访问 https://nodejs.org/
2. 下载 LTS 版本（建议 16.0.0 或更高）
3. 按默认设置安装

验证安装：
```bash
node --version
```

### 1.3 下载项目文件

**方式一：直接下载（推荐新手）**
1. 访问 https://github.com/NanmiCoder/MediaCrawler
2. 点击绿色的 "Code" 按钮
3. 选择 "Download ZIP"
4. 下载后解压到任意文件夹

**方式二：使用 Git 克隆**
```bash
git clone https://github.com/NanmiCoder/MediaCrawler.git
cd MediaCrawler
```

---

## 🚀 第二步：安装和启动

### 2.1 使用 uv 安装依赖（推荐方式）

在项目文件夹中打开命令行：

```bash
# 进入项目目录
cd MediaCrawler

# 使用 uv sync 自动管理虚拟环境和依赖
uv sync

# 安装浏览器驱动
uv run playwright install
```

> ⏰ **等待时间**：首次安装可能需要 10-15 分钟，uv 会自动：
> - 创建虚拟环境
> - 安装 Python（如果需要）
> - 安装所有依赖包
> - 请耐心等待

### 2.2 安装 Web 界面依赖

```bash
# 安装 Web 管理界面的额外依赖
uv pip install fastapi uvicorn psutil pydantic
```

### 2.3 启动 Web 界面

```bash
# 使用 uv 启动 Web 服务
uv run python start_web.py
```

看到以下信息说明启动成功：
```
🚀 启动 MediaCrawler Web 服务...
📊 管理界面: http://localhost:8000
📖 API 文档: http://localhost:8000/docs
```

浏览器会自动打开管理界面，如果没有自动打开，请手动访问：
**http://localhost:8000**

## 🎮 第三步：使用 Web 界面

### 3.1 界面介绍

登录后您会看到 4 个主要功能页面：

| 图标 | 页面名称 | 功能说明 |
|------|----------|----------|
| 📊 | 数据概览 | 查看爬取的数据统计和图表 |
| 🎮 | 爬虫控制 | 启动/停止爬虫，设置爬取参数 |
| 📋 | 数据管理 | 浏览、搜索、导出爬取的数据 |
| ⚙️ | 配置管理 | 修改爬虫的各种设置 |

### 3.2 第一次爬取数据

**步骤 1：点击"爬虫控制"页面**

**步骤 2：填写爬取参数**
- **选择平台**：建议新手先选择"小红书"（相对稳定）
- **爬取类型**：选择"关键词搜索"
- **关键词**：输入您想搜索的内容，如："美食推荐"
- **登录方式**：选择"二维码登录"（最简单）
- **爬取评论**：建议先关闭（加快速度）
- **最大数量**：建议设置为 20（测试用）
- **数据保存方式**：选择"SQLite数据库"

**步骤 3：启动爬虫**
1. 点击"启动爬虫"按钮
2. 会弹出二维码窗口
3. 用手机打开对应 APP（如小红书）
4. 扫描二维码登录
5. 等待爬取完成

**步骤 4：查看结果**
1. 爬取完成后，点击"数据管理"页面
2. 您可以看到爬取到的帖子数据
3. 点击"详情"可以查看具体内容
4. 点击"导出"可以保存为 CSV 文件

## 📊 第四步：查看和分析数据

### 4.1 数据概览页面

- **统计卡片**：显示总帖子数、评论数等
- **平台分布图**：饼图显示各平台数据占比
- **活动趋势图**：折线图显示最近的爬取活动
- **最近活动**：列表显示最新的爬取记录

### 4.2 数据管理页面

**搜索功能**：
- 选择特定平台查看数据
- 输入关键词搜索相关帖子
- 选择时间范围筛选数据

**查看详情**：
- 点击"详情"按钮查看完整帖子内容
- 点击"评论"按钮查看用户评论
- 支持查看图片和视频链接

**导出数据**：
1. 选中要导出的数据行
2. 点击"导出"按钮
3. 文件会自动下载到您的电脑

## ⚙️ 第五步：个性化配置

### 5.1 基础配置

在"配置管理"页面，您可以：
- 设置默认平台和爬取类型
- 修改默认关键词
- 调整数据保存方式

### 5.2 爬取配置

- **最大帖子数量**：建议新手设置为 50-200
- **爬取间隔**：建议设置为 2-5 秒（避免被限制）
- **是否爬取评论**：会显著增加时间，按需开启
- **是否下载媒体文件**：会占用大量存储空间

### 5.3 浏览器配置

- **无头模式**：关闭后可以看到浏览器操作过程
- **保存登录状态**：开启后下次无需重新登录
- **CDP 模式**：使用本地 Chrome 浏览器（更稳定）

## 🔧 常见问题解决

### Q1: uv 安装失败
**解决方案**：
```bash
# 方法一：使用 pip 安装
pip install uv

# 方法二：手动下载安装
# 访问 https://github.com/astral-sh/uv/releases
```

### Q2: uv sync 失败
**解决方案**：
```bash
# 清除缓存重试
uv cache clean
uv sync

# 或使用传统方式
python -m venv venv
# Windows
venv\Scripts\activate
# Mac/Linux  
source venv/bin/activate
pip install -r requirements.txt
```

### Q3: 二维码登录失败
**解决方案**：
1. 确保手机和电脑在同一网络
2. 尝试关闭无头模式，手动处理验证
3. 更换登录方式为"Cookie登录"
4. 启用 CDP 模式使用本地浏览器

### Q4: 爬取速度很慢
**解决方案**：
1. 减少爬取数量
2. 增加爬取间隔时间
3. 关闭评论爬取
4. 检查网络连接
5. 启用 CDP 模式

### Q5: 浏览器启动失败
**解决方案**：
```bash
# 重新安装浏览器驱动
uv run playwright install chromium

# 或启用 CDP 模式使用系统浏览器
# 在配置管理中开启 "启用CDP模式"
```

## 📚 进阶使用技巧

### 1. 命令行方式爬取
除了 Web 界面，您也可以直接使用命令行：

```bash
# 使用 uv 运行（推荐）
uv run main.py --platform xhs --lt qrcode --type search

# 或在虚拟环境中运行
python main.py --platform xhs --lt qrcode --type search
```

### 2. 批量关键词爬取
在关键词输入框中用英文逗号分隔多个关键词：
```
美食推荐,旅游攻略,数码测评
```

### 3. 使用 CDP 模式（推荐）
- 在"配置管理"中启用"CDP模式"
- 使用您本地的 Chrome/Edge 浏览器
- 更稳定，不容易被检测

### 4. 数据分析建议
1. 导出数据到 Excel 进行深度分析
2. 关注点赞数和评论数的关系
3. 分析发布时间和互动量的关系
4. 研究热门内容的共同特征
